
## Laravel 5 - Typehead

Laravel 5, Mysql ve Twitter Typehead ile çalışan otomatik arama örneği

### Kurulum işlemleri
- `composer install` komutu ile gereksinimleri kurun
- `.env` dosyasındaki veritabanı bağlantısı bilgilerini girin
- `php artisan key:generate` 
- `php artisan migrate` 
- `php artisan db:seed`
